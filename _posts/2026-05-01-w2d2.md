### 引入拓扑排序


``` python
class Value:
    # ... 之前的代码 (__init__, __add__, __mul__ 等) ...

    def backward(self):
        # 1. 寻找家谱并排序
        topo = []
        visited = set()  # 对所以的Value对象去重
        def build_topo(v): # 创建一个topo函数，让所有参与计算的数据都按顺序出现在列表里
            if v not in visited:
                visited.add(v)
                for child in v._prev: # 当字母没有_prev时，for循环不会执行
                    build_topo(child) # 运行刚定义的函数，所有的v都会变成child进行
                topo.append(v)
        build_topo(self)

        # 2. 触发骨牌效应
        self.grad = 1.0 # 钦定最后的输出的梯度为 1
        for node in reversed(topo): # 将topo从正向顺序逆转成反向顺序。
            node._backward() # 运行_backward函数，求出所有对象的梯度

```
### 引用tanh函数将所有输入压缩到0～1之间


``` python
import math

class Value:
    # ... 之前的代码 ...

    def tanh(self):
        x = self.data
        t = (math.exp(2*x) - 1)/(math.exp(2*x) + 1)
        out = Value(t, (self, ), 'tanh')
        
        def _backward():
            # 数学推导：tanh(x) 的导数是 1 - tanh(x)^2
            # 这里的 t 就是 tanh(x)
            self.grad += (1 - t**2) * out.grad
        out._backward = _backward
        
        return out
```

### 搭建神经元

![w2d2-1777612142599.webp](/images/w2d2-1777612142599.webp)
什么是神经元？神经网络中的输入层，隐藏层，输出层的所有小圆圈都是神经元，神经元接收多个输入信号，经过一系列处理输出信号。这个处理过程通常分为两步：

1. **加权求和**：每个输入 $x$ 都有一个对应的权重 $w$。我们要把所有的 $x \times w$ 加起来，最后再加上一个偏置项 $b$（Bias）。
    
    - 公式表达：$z = \sum (w_i \cdot x_i) + b$
    
2. **激活处理**：为了让网络具备处理复杂问题的能力，我们会把 $z$ 丢进刚才讨论过的 `tanh` 函数里。
![w2d2-1777614589083.webp](/images/w2d2-1777614589083.webp)
代码表示：

```python
import random

class Neuron:
    # nin 代表 "number of inputs"（输入信号的数量）
    def __init__(self, nin):
        # 随机初始化权重和偏置，让神经元一开始有不同的“个性”
        self.w = [Value(random.uniform(-1, 1)) for _ in range(nin)]
        self.b = Value(random.uniform(-1, 1))
        
    def __call__(self, x): # 只要在对象名后面加上()就会自动运行。
        # 实现 w * x + b
        # zip(self.w, x) 把权重和输入一对一配对
        act = sum((wi*xi for wi, xi in zip(self.w, x)), self.b)
        out = act.tanh()
        return out
```


### 构建“层”--Layer
层和神经元的关系：层是由多个神经元构成。每一层里的多个神经元接收信息，输出多个结果，比如：接收3个输入，输出4个结果。


``` python
class Layer:
    def __init__(self, nin, nout):
        # 创建 nout 个神经元，每个神经元都有 nin 个输入
        self.neurons = [Neuron(nin) for _ in range(nout)]
    
    def __call__(self, x):
    
        # 让每个神经元都算一遍，把结果存在列表里，n(x)调用了Neuron里的call
        outs = [n(x) for n in self.neurons]
        
        # 如果只有一个神经元，我们就直接把结果拿出来，不用包在列表里了
        return outs[0] if len(outs) == 1 else outs
```

### 多层感知机--MLP

顾名思义--把所有层结合起来，上一层有多少个神经元，下一层的每个神经元就有多少个输入。

``` python
class MLP:
    def __init__(self, nin, nouts):
        # nouts 是一个列表，比如 [4, 4, 1] 表示三层，每层神经元数量不同
        sz = [nin] + nouts # nin指的是最先输入到神经元里信息的数量
        self.layers = [Layer(sz[i], sz[i+1]) for i in range(len(nouts))]

    def __call__(self, x):
        # 像传声筒一样，把输入一层层传下去
        for layer in self.layers:
            x = layer(x)
        return x
```
