## makemore-1

更加深入的了解大语言模型训练的过程，从简单的名字预测开始。引入一个含有三万多名字的文本text--https://github.com/karpathy/makemore/blob/master/names.txt

``` python
# 在python中打开并读取,splitlines()表示分行显示
words = open('names.txt', 'r').read().splitlines()
```
![w2d4-1777788112778.webp](/images/w2d4-1777788112778.webp)

``` python
b = {}
for w in words:
  chs = ['<S>'] + list(w) + ['<E>'] # 给所有的名字都加一个前后缀。
  for ch1, ch2 in zip(chs, chs[1:]): # 对每个名字中连续的两个字母
    bigram = (ch1, ch2)
    b[bigram] = b.get(bigram, 0) + 1 # 记录所有连续两个字母出现的次数
    
sorted(b.items(),key = lambda kv:-kv[1]) # 对次数进行按次数从大到小的排序
```

## 引入pytorch张量


``` python
import torch

N = torch.zeros((27, 27), dtype=torch.int32)# 27行27列的0矩阵

chars = sorted(list(set(''.join(words)))) # 按从a～z列出所有的字母
stoi = {s:i+1 for i,s in enumerate(chars)}# 对字母进行序号标志
stoi['.'] = 0 # .用序号0表示
itos = {i:s for s,i in stoi.items()} # 序号和字母互换位置

for w in words:
  chs = ['.'] + list(w) + ['.']
  for ch1, ch2 in zip(chs, chs[1:]):
    ix1 = stoi[ch1] # 将字母转化为数字，用于之后的数学计算
    ix2 = stoi[ch2]
    N[ix1, ix2] += 1 # 计算连续的两个字母出现的次数
```

## 创建随机数生成器


``` python
 # 为了让实验结果可复现，我们会固定随机种子。只要种子相同，每次运行代码生成的“随机”序列都是完全一样的。
g = torch.Generator().manual_seed(2147483647)

# 接收概率张量，也就是p，随机取样（这里是20），有放回抽取，使用我们定义的g
ix = torch.multinomial(p, num_samples=20, replacement=True, generator=g).item()

itos[ix] # 输出d

```
