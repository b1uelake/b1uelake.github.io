# rand时pytorh的内置函数，用于生成均匀分布的随机数，我们将均匀分布划分为3个结果。

 `torch.multinomial`是 PyTorch 的一个内置函数，用于进行**多项式分布采样**。按照概率抽取位置，该位置的概率越大，就越容易抽到。

```python
# 由于有些连续两个字母出现的次数为0，会导致后续计算时出现问题，所以统一对所有出现次数都+1，P最后是概率，我们需要转化为浮点数。
P = (N+1).float()

# 1代表维度，只将一行的和相加，keepdims=True是保持维度
P /= P.sum(1, keepdims=True)

g = torch.Generator().manual_seed(2147483647)

# 随机生成5个预测的名字
for i in range(5):
  
  out = []
  ix = 0
  while True:
    p = P[ix]
    ix = torch.multinomial(p, num_samples=1, replacement=True, generator=g).item()
    out.append(itos[ix])
    if ix == 0:
      break
  print(''.join(out))
  # 输出：cexze. momasurailezitynn. konimittain. llayn. ka.
  
```

### 用对数计算概率
因为所有概率都小于0，计算联合概率时把所有概率相乘会输出一个特别小的数，使浮点数溢出。但是转化为log后，概率相乘等于log（）外相加。
log(a*b*c) = log(a) + log(b) + log(c)
``` python
for w in words:
#for w in ["andrejq"]:
  chs = ['.'] + list(w) + ['.']
  for ch1, ch2 in zip(chs, chs[1:]):
    ix1 = stoi[ch1] # 取数
    ix2 = stoi[ch2]
    prob = P[ix1, ix2] # 查概率
    logprob = torch.log(prob) # 计算对数概率
    log_likelihood += logprob # 将对数概率相加
    n += 1 # 记录处理了多少次

```

### 引入pytorch中的独热编码
网址介绍：https://docs.pytorch.ac.cn/docs/stable/generated/torch.nn.functional.one_hot.html
作用：将数字的对应位置的0变成1，用于矩阵计算。
``` python
import torch.nn.functional as F
xenc = F.one_hot(xs, num_classes=27).float()
xenc
```

其中每一个数字代表出现的可能性的得分，正数可能性大，负数可能性小。
我们希望他们转化为概率，这样能够更直观

```python
logits = xenc @ W # log-counts
counts = logits.exp() # equivalent N
probs = counts / counts.sum(1, keepdims=True)

```
