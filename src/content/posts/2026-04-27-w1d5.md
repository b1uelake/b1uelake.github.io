## 前置知识
### python中的‘类’

先看例子：![w1d5-1777296890288.webp](/images/w1d5-1777296890288.webp)
这是三个字典，stu1,stu2,stu3称为‘对象’ ，name,sex,age,height称为‘属性’,
后面的具体信息称为‘数据’。
假设班上有50个学生，他们的数据都不同，如果每次都类似这样创建，就很麻烦。
此时我们使用‘类’来化繁为简。首先我们看到，对象的属性相同，都包含-name,sex,age,height.只是后面的数据不一样。

此时我们就可以创建一个类--class

```
class Student:
    def __init__(self, name, sex, age, height):
        self.name = name
        self.sex = sex
        self.age = age
        self.height = height
# 根据图片内容实例化对象，类会自动识别name，sex，age，height。
stu1 = Student('张三', '男', 16, 167)
stu2 = Student('李四', '女', 15, 160)
stu3 = Student('王五', '女', 14, 163)

# 示例：访问数据
print(stu1.name)  # 输出: 张三
print(stu2)       # 输出: Student(name='李四', sex='女', age=15, height=160)
```
![w1d5-1777291708062.webp](/images/w1d5-1777291708062.webp)
 `__init__`的意思是初始化，在类中必须有，括号中：self泛指所有对象（必须有），剩下的就是属性的命名。
self.xxx=a 表示用a给属性赋值。
